# Cucumber Archetype

This is a Maven Archetype for setting up an empty Cucumber project.

```shell
mvn archetype:generate                                  \
  -DarchetypeGroupId=io.cucumber                        \
  -DarchetypeArtifactId=cucumber-archetype              \
  -DarchetypeVersion=4.0.0.0                            \
```
